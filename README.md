# Sportivniy-komplex

## О проекте

Приложение позволяет пользователям записываться на занятия, а администраторам управлять списком

## Основные функции

- Просмотр тренировок 
- Просмотр участников
- Добавление тренировок

## Технологии

- React.js + Redux  
- Node.js + Express  
- PostgreSQL  
- Socket.IO для реального времени  
- Bootstrap / Tailwind CSS
